# Text Review & Critique

## Description
Precise text review that identifies specific problems with line numbers and solutions.

## Use Case
For detailed review of documents, articles, or any written content requiring critical feedback.

## Instructions

[text]

Find:
- Unclear parts (line number)
- Claims without support
- Logic gaps

Don't give me generic feedback.
Line number + problem + fix.

## Tips
- Provides actionable, specific feedback
- No generic advice
- Good for academic or technical writing
- Request focus on specific aspects if needed
